🏢 CondoCombat

Porque planilha de briga de vizinho
é humilhante — sua treta merece um sistema.

🏗️ Landing (Astro) → 🖥️ App (Next.js) → ⚙️ API (FastAPI)
O único ERP que trata fofoca de elevador como caso de uso legítimo.

🔥 O Problema

Condomínios brasileiros são microcosmos de conflitos:
barulho, fofoca, disputas por vaga, obras irregulares,
animais soltos, síndicos autoritários, inadimplência…

❌ Planilhas perdidas

❌ Grupos de WhatsApp tóxicos

❌ E-mails que ninguém lê

Não existe ferramenta dedicada para gerenciar o caos.

💡 A Solução

em dado estruturado desde 2026.

🥞 Stack Tecnológica

Landing Page
Astro + Tailwind
HTML estático, zero JS, máxima performance
Frontend App
Next.js + shadcn/ui
App Router, Server Components, tipagem estrita
Backend API
FastAPI + SQLAlchemy

TypeScript (front) · Python (back)
